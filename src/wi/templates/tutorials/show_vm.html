Here is list of your virtual machines. You can manage and see details of any VM.

VM managment actions

  • Destroy - irreversibly destroyes the VM without saving changes to image.
  • Save and shutdown - closes the VM and saves the image with changes in user's private images pool.
  • Reset - unplugs the VM and connects it to power again. Unsaved changes on the machine will be lost.
  • Edit - edits VM's name and description. If VM with contextualization is closed from inside, it will be saved to image taking that name with 'autosave' appended.

Resources

VM access actions

  • Assign external IP - allows access to the VM from the internet.
  • Graphical console (VNC) - transmits the VM screen to the webbrowser window.

VM contextualization actions (available only if VM supports contextualization)

  • Set password - set user and password on the VM.
  • Set SSH key - set SSH key on the VM.
