Farm creation form is divided into four steps analogous to VM creation:

  • Image choice: You select one common image for all nodes and for head. That makes them start with the same configuration.
  • Hardware choice: However, you can select diffrent template for head and for nodes, giving them diffrent computational power.
    • Head template
    • Worker node template
    • Number of worker nodes
  • Optional resources: It is recommended to attach disk volume to store data on it. This is only possible on stage of farm creation. You are not able to attach disk to already created farms.
  • Confirmation
