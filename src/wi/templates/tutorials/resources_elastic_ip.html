Here you can manage your elastic external IP adressess. Attaching public IP to your VM makes it accessible from any place in the world.

By default one user can obtain one public IP address due to limited public IP pool.

You can configure your VMs so that you have access to all of them having only one IP address (e.g. using VPN).

Available actions:

  • Add new IP - available, if you haven't used your limit.
  • Release IP - removes corresponding IP address from your pool.
